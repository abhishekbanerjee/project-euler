# Project Euler
Solving [Project Euler](https://projecteuler.net) problems, one commit at a time.

![Project Euler](https://projecteuler.net/profile/abhi_banerjee.png)

## Run the Rust Code

```shell
$ cd euler-rust
$ cargo build
$ cargo run --bin problem-00001
```
